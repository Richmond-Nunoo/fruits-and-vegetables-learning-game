# fruits-and-vegetables-learning-game
A simple kids learning game to teach fruits and vegetables using emojis, flashcards, and basic quizzes. Built with pure HTML, CSS, and vanilla JavaScript.



- Flashcards (Learn Mode)  
- Quiz Mode (Multiple Choice)  
- Spelling Practice Mode  
- Text-to-speech voice playback  
- Cute food emojis

##  Features

- 3 learning modes (Learn / Quiz / Spell)
- Voice support using Web Speech API
- Simple and fun UI for kids
- Sound feedback for right/wrong answers
- Category filter (Fruits / Vegetables / All)

##  How to Use

1. Select category (Fruits / Vegetables / All)
2. Choose Mode:
   -  Learn: See and hear names  
   -  Quiz: Pick the right answer  
   -  Spell: Type the word  
3. Have fun learning!
